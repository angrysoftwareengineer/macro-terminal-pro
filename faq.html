FAQ

Whether the purchase is indefinite or has a validity period.

The purchase is for an unlimited period of time and has no expiry date.

Graphics (simulator) is unresponsive.

Press the reset button in the upper right corner.

The message "Program not found" appears.

You are probably using the wrong settings - for example: if you want to create a program for a Haas machine, select the Haas option in the settings panel. If it doesn't work, please contact me.

"R" value in G2 or G3 is not working properly.

The CNC terminal macro does not yet support the "R" value for circular interpolation.

I found an error, what should I do?

If you found any bug, please contact me.

I can't save the file.

On your device, go to settings, find “apps”, then find Macro Terminal CNC, select "allowed storage". If it hasn't worked, please contact me.

After pressing the "Exec" button (the light gray triangle in the upper left corner of the upper bar) the application freezes.

Probably the program you are trying to execute loops endlessly. If you can't find the wrong part in your code, go to the "exec line" and press "Play" or "Play fast".After a while you should recognize what part of the code is repeating.If You are still not sure, press “Stop” and then manually step through the lines using the button “Next”.

Why does the CNC Macro Terminal not have 3D graphics?

The mobile phone has limited resources and, moreover, the screen size is too small to manipulate the 3D graphics with your fingers.
