Overview

Macro Simulator CNC is a powerful, cross-platform CNC code editor, CNC simulator and backplotter with conversational shell for G-Code (RS-274) and Fanuc Macro B/Haas/Centroid programs (CNC parametric programming) for 3-axis CNC milling machines.

The first version of Macro Simulator CNC called Macro Terminal CNC was only available for mobile devices. However, it quickly gained great success and great reviews, especially in the US market, which was, among other things, due to the innovative cnc macro analysis tool that allows you to see all logical and arithmetic evaluations, one by one. This success and the demand for similar software for stationary devices resulted in the creation of Macro Simulator CNC. Therefore, this software is now available for many operating systems and supports both stationary and mobile devices.
