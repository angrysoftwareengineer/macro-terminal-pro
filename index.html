Macro Simulator CNC (successor to Macro Terminal CNC) is a powerful, cross-platform CNC code editor, CNC simulator and backplotter with a micro command system (special keywords and cycles)* and an Advanced Error Recognition System (AERS) for G-Code (RS-274) and Fanuc Macro B programs (parametric programming) for 3-axis CNC milling machines.

Macro Simulator CNC was created primarily to improve CNC parametric programming (using CNC macros) with an extremely powerful macro debugging tool, showing the math and logical calculations calculated during macro program execution. Now, however, it has gained a new functionality: it can use different types of milling cycles and drilling patterns, using a special syntax: micro-commands that, after converting, work on all machines that support the ISO (G-code) standard!
