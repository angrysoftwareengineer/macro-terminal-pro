What's SETVN?

SETVN, or Set Variable Name, is a command used in Fanuc CNC controls, specifically within the context of Custom Macro B programming. This command allows users to assign a descriptive name (up to eight characters) to certain permanent common variables (specifically the #500 series variables). This naming feature enhances the documentation and readability of programs by providing meaningful labels next to these variables when displayed on the control's screen.

Functionality of SETVN

  • Purpose - The primary function of SETVN is to document the use of permanent common variables by allowing programmers to assign human-readable names to them. This helps avoid confusion when multiple programs interact with these variables.
  • Variable Range - The command can be applied to a limited number of permanent common variables, typically from #500 to #549, depending on the specific Fanuc control model. Some models may allow documentation for the first ten (#500 through #509), while others may extend this capability up to fifty variables.

Command Format

The syntax for using the SETVN command is as follows:
SETVN <variable_number> [<name>, <name2>, <name3>]
For example:
SETVN 500 [DIAMETER, ANGLE, ANGLE]
would assign the names #DIAMETER, #ANGLE, #ANGLE to the variables #510 through #512 respectively. This allows for easier identification of what each variable represents in the program

Good to know

In this example, "DIAMETER" will appear next to the variable #500 on the display, providing context for its use
