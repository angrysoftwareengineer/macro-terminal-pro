Technical information

Conversational mode (only conversational version):

  • POCKET - circular/rectangular pocket milling
  • CYCLE - simple drilling, dwell drilling, peck drilling, taping
  • THREAD MILLING - external and internal thread milling
  • SURFACE - face milling (3 strategies)
  • SHOULDER - shoulder circular or rectangle milling, circular or rectangle grooving
  • DRILLING - peck drilling with dwell time (between pecks) and full retract (between pecks)
  • HELICAL BORING - helical boring, circular pocket milling

Micro commands (only conversational version):

  • TOOL - tool call
  • SET - changing global data used by other commands such as approach point (X, Y, Z), rotation (A), coolant (C), retraction (R), spindle direction (D), work offset (W) or surface coordinate (S)
  • CYCLE - drilling / tapping cycles
  • POCKET - pocket milling (circular and rectangle) and groving milling (circular and rectangle)
  • SHOULDER - shoulder milling (circular and rectangle)
  • SURFACE - face milling (3 strategies)
  • PATTERN - drilling patterns (4 strategies)
  • THREADING - milling threading external and internal
  • DRILLING - peck drilling with dwell time (between pecks) and full retract (between pecks)
  • HELICAL BORING - helical boring
  • MIRROR - reverse coordinates in x, y axis
  • ZOOM - resize coordinates x and y
  • SHIFT - move coordinates in x, y axis
  • ROTATE - rotate coordinates in x, y axis

Program export:

  • code compressing
  • removing empty lines
  • adding decimal point
  • adding automatically information about the tool used (at the beginning of the program as before calling it)
  • adding line number
  • adding safety line
  • auto adding end of program (M30)

Viewport (simulator/graphics) information:

  • 3D tool path
  • simulation (play, play fast, stop, next, reset, previous, rewind)
  • 5 view modes (3D, XY, XY removal, XZ, YZ)
  • image rotation in x axis
  • zoom
  • shift image
  • show/hide rapid movement
  • simplified representation of tools (max 50 tools)
  • metric/imperial mode
  • 3 themes (dark, bright, black & white)

G-code/Fanuc/Haas/Mazak ISO support:

  • absolute, incremental and polar mode
  • G16 polar coordinate
  • G51 scaling
  • G52 local work shift
  • G65/G66 macro subprogram call
  • G68 coordinate system rotation
  • G98 retract to initial Z value
  • G99 retract to R value
  • M98 subprogram (multiple formats)
  • repeat canned cycle (L and K)
  • circular interpolation supported by using (I, J or R)

Haas support:

  • G70 Bolt Hole Circle
  • G71 Bolt Hole Arc
  • G72 Bolt Holes Along an Angle

Macro (Fanuc Custom Macro B/Haas/Centroid) support:

  • keywords: IF, THEN, GOTO, WHILE, DO, END, ELSE
  • comparison operators (FANUC/HAAS): EQ, LT, LE, GT, GE, NE
  • comparison operators (CENTROID): == or EQ, < OR LT, <= or LE, > or GT, >= or GE, != or NE
  • math functions supported: sin, cos, tan, asin, acos, atan, sqrt, abs, round, fix, fup, ln, exp, pow
  • local, common variables and some of the most popular system variables
  • support for both syntax FANUC/HAAS and CENTROID
    • IF [ #100 EQ 200 ] THEN GOTO 15 (FANUC MACRO B SYNTAX)
    IF #100 == 200 THEN GOTO 15 ELSE GOTO 55 (CENTROID SYNTAX)

Debugger:

  • assigned and created variables
  • math and logical calculations in line
  • for IF and WHILE statement shows the actual value (true or false)

Code editor:

  • code highlighting
  • formatting
  • undo/redo
  • adding/removing EOB
  • adding/removing %
  • adding decimal point

Files:

  • new project
  • open file
  • open template
  • append(to add a subroutine locally)
  • save
  • save as
  • export(to export macro program as a pure g code program)
  • file extension supported:nc, cn, ncc, cnc, eia, txt

Input compatibility:

Fanuc 0, Fanuc 6, Fanuc 10, Fanuc 11, Fanuc 12, Fanuc 15, Fanuc 16, Fanuc 18, Fanuc 20, Fanuc 21, Fanuc 30i, Fanuc 31i, Fanuc 32i, Fanuc 35i, Haas, Centroid, Mazak ISO, Siemens 802D and others using CNC macros based on Fanuc Custom Macro B.

Output compatibility:

Same as input control selected, it also can be converted to Heidenhain, Sinumerik or other control upon special request.

Languages:

English, Français, Deutsche, Italiano, Español, 한국어, 日本語, Polski, Magyar, Pусский, Português, 中文, Türkçe
